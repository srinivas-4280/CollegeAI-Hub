## ⚡ Tech Stack

`HTML5` · `CSS3` · `JavaScript` · `Groq API` · `Google Gemini API` · `LocalStorage`

> Single HTML file. Zero frameworks. Zero dependencies.

## 🚀 Setup

1. Clone or download this repository
2. Open `index.html` in your browser
3. Get a free Groq API key
4. Paste key → Save → Start using tools

## 🔒 Privacy

API keys are stored locally in your browser. No server, no tracking.
